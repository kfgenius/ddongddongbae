v1.1c - 수정 내용

1. 잘못된 주석 처리 수정
이전에 "http://hondoom.com"이라고 쓰면 // 이후를 주석으로 인식했는데
이번 버전부터는 ""안의 //는 무시.
단, 흥크립트 1.0b에서는 지원하지 않으므로 txt파일 형태에서는 에러가 남.

2. 글꼴 명령 추가
@글꼴 "글꼴명" 식으로 글꼴 변경 가능.
단 두가지 문제가 발생하니 유의할 것.
1) 자동 줄바꿈이 지원 안 된다. 강제 줄바꿈으로 잘 적으면 괜찮다.
2) 상대방의 컴퓨터에 글꼴이 없을 경우 기본 글꼴로 나온다.

3. 파란색이 무조건 투명되는 것 수정
이제 다른 색도 ChunkMaker에서 설정된 값으로 투명색을 쓸 수 있음

4. 시스템 변수 얻기
|시스템 변수명| 식으로 적으면 예약된 시스템 변수명을 가져 올 수 있다.
쓰기는 불가능하고 읽기만 가능하다.

|선택| : 선택지에서 선택한 번호. 주의할 것은 첫번째 선택지는 1이 아니라 0이라는 것.
         응용하면 8개의 선택지까지 가능한 한계를 극복할 수 있다.
|커서명| : 영역 선택시 현재 지정된 커서이름을 얻는다.
|커서X| : 영역 선택시 커서의 X좌표를 가져온다.
|커서Y| : 영역 선택시 커서의 Y좌표를 가져온다.
|파일명| : 현재 읽고 있는 스크립트 명을 가져온다.
|챕터명| : 현재 챕터명을 가져온다.
|배경음| : 현재 배경음 파일명을 가져온다.
